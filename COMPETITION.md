## 一、作品名称

**虾评爆点 · Lobster Hot Take**

## 二、解决什么问题

内容创作者/自媒体人每天面临三重痛点：

1. **找梗难** — 每天刷 5+ 平台找热点，耗时 1-2 小时
2. **做图慢** — 从热点到梗图需要构思文案 + 作图 + 排版，30 分钟/张
3. **无记忆** — 每次从零开始，不知道粉丝喜欢什么风格

**虾评爆点**用一条命令解决全部三个问题：

```bash
python -m meme_hunter
# 30 秒后：5 张梗图 + 建筑极简风梗图墙 + 一键拷贝分发
```

## 三、核心创意：虾总 IP + 自循环

### 3.1 虾总是谁？

一只拟人化红色龙虾，名叫「虾总」。
它会**根据每条热点新闻自动切换角色**（CEO、程序员、HR、00后实习生……），
用钳子做出与梗点呼应的动作，自带喜感。

> 为什么用龙虾：本 Skill 兼容 OpenClaw（意为"爪"）平台，
> 龙虾 = Claw = 品牌呼应。

### 3.2 自循环越跑越准

用户不只是被动看梗图，而是通过**两种反馈闭环**教虾总你的笑点在哪：

**闭环 A：首次笑点测试（8 题）**
```
新用户进入 → 弹出虾总叙事式测试 → 答完生成笑点画像 → 写入 soul.md
→ 后续所有梗图按画像方向倾斜
```

**闭环 B：持续点赞反馈**
```
看梗图 → 点「戳到我了」→ 写入 soul.md liked 列表
→ 每 5 次自动触发 LLM reflect → 重新分析画像维度 + 提取风格关键词
→ 下一轮生成注入新 hint → 越来越懂你
```

**soul.md 格式兼容 OpenClaw SOUL.md 规范**，可跨平台迁移用户画像。

## 四、使用的 Wan 能力

| Wan2.7 能力 | 用途 | 调用方式 |
|---|---|---|
| **文生图** `wan2.7-image` | 根据梗点 prompt 生成虾总主体图 | `ImageGeneration.call()` |
| **序列一致** `enable_sequential=True` | 保证同一批梗图中虾总外形一致 | 同上，n>1 时自动启用 |

> Wan2.7 是本 Skill 的**唯一图像引擎**，所有图片均由 Wan2.7 生成。

## 六、Prompt 工作流

### 步骤 1：热点 → 梗点

```
[System] 你是「虾评爆点」的段子手 + 虾总 IP 人设
[User]   热点：某AI公司裁员50%，CEO切蛋糕庆生
[Output] JSON: {meme_point, image_prompt, top_text, bottom_text}
```

### 步骤 2：梗点 → Wan2.7 prompt

```
拟人龙虾虾总，红色硬壳发亮，圆眼睛左笑右眯，
戴金丝眼镜和黑色高定西装+红领带，
左手高举插着'战略聚焦'蜡烛的双层奶油蛋糕，
右手正将一叠写有'50%'的A4纸塞进冒烟的碎纸机；
背景是极简科技风办公室；电影感侧逆光，皮克斯3D渲染质感
```

### 步骤 3：Pillow 后处理

```
底图 → 顶部黑条(上联) → 底部黑条(下联) → 暖色分割线 → 来源水印
```

## 九、差异化亮点

1. **虾总 IP** — 市面无同类，龙虾 × 热点 × 皮克斯风格 = 高辨识度
2. **自循环** — 不是一次性 demo，而是越用越准的个人笑点引擎
3. **一个 Key** — DashScope Key 同时驱动 Wan2.7 + Qwen，普通用户 10 分钟上手
4. **4 种模式** — CLI / Web / Skill / OpenClaw，覆盖所有用户群
5. **建筑极简风** — 截图即海报，自带传播属性
6. **开源** — MIT 协议，评委可 fork 当场跑
