02 / PRODUCT SCREENSHOTS

产品界面

建筑极简风设计，黑白 + 暖色点缀，截图即海报。

设置页 — 填入 DashScope API Key，选择 LLM Provider（6种可选）， 勾选热点来源（微博/知乎/IT之家/抖音/36氪/百度/B站），点击「让虾总开工」
梗图墙 — 虾总出品的梗图自动排列，每张带来源标记 + 原文链接 + 一键复制文案 + 下载图片 + 「戳到我了」点赞反馈
03 / HOW IT WORKS

从热搜到梗图，全自动

STEP 01
抓取全网热点
同时采集微博热搜、知乎热榜、IT之家、抖音、36氪、百度、B站 7 大平台热点
STEP 02
LLM 改编梗点
Qwen / DeepSeek / MiniMax 等 6 种 LLM 将热点改编为上下联 + 虾总角色 + 画面描述
STEP 03
Wan2.7 文生图
通义万相 Wan2.7 根据 prompt 生成虾总出镜的梗图主体，支持序列一致性
STEP 04
Pillow 烧字幕
自动添加上联、下联、暖色分割线、来源水印，输出成品梗图
STEP 05
建筑极简风展示
生成梗图墙 HTML，支持一键复制文案、下载图片、点赞反馈写入 soul.md
